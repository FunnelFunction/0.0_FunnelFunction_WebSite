## Running Tests

### All Tests
```bash
# Python tests
pytest 9.0_tests/

# JavaScript tests
npm test
```

### Unit Tests Only
```bash
pytest 9.0_tests/9.1_tests_unit/
```

### Integration Tests
```bash
pytest 9.0_tests/9.2_tests_integration/
```
