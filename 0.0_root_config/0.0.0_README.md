# Funnel Function Institute - Website Repository

> **The Mathematical Architecture of Commercial Attention**

---

## Repository Purpose

This repository serves as the **structural backbone** for the Funnel Function Institute's web presence. It contains:

- Source content for all website pages
- Mathematical equation implementations (Python, PyTorch)
- Simulator source code
- Social media aggregation systems
- Build and deployment scripts

---

## Architecture Overview

```
WordPress (funnelfunction.com)     GitHub (this repo)
─────────────────────────────     ──────────────────
Marketing site / CMS         ←──→  Source of truth
Live content delivery              Version control
SEO / Analytics                    Code / Equations
                                   Simulators
```

**WordPress** handles content delivery and SEO.
**GitHub** handles version control, code, and structured content.

## Naming Convention

```
[MAJOR].[MINOR].[PATCH]_[type]_[descriptor].ext
```

**Type prefixes:**
- `index_` = Entry point / table of contents
- `content_` = Prose / written content
- `config_` = Configuration file
- `script_` = Executable script
- `embed_` = Embeddable HTML widget
- `styles_` = CSS styling
- `util_` = Utility function
