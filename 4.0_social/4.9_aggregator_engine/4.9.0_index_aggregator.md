# Aggregator Engine

> **4.9_aggregator_engine** - Master social media aggregation system

---

## Purpose

The aggregator engine:
1. Fetches content from all social platforms
2. Extracts keywords and topics
3. Generates SEO-friendly content
4. Updates the /social/ page

## Keyword Extraction

The SEO keyword generator uses:
- TF-IDF for term importance
- Named Entity Recognition for proper nouns
- Custom domain vocabulary (marketing, math terms)

Output format:
```json
{
  "keywords": ["f(Commitment)", "identity congruence", "marketing mathematics"],
  "topics": ["computational marketing", "trust decomposition"],
  "entities": ["Ryan Reynolds", "Mint Mobile", "Funnel Function Institute"],
  "hashtags": ["#MarketingScience", "#FunnelFunction"]
}
```
