## Configuration

```json
{
  "platform": "twitter",
  "handle": "@funnelfunction",
  "api_version": "v2",
  "refresh_interval": "1h",
  "extract_keywords": true,
  "max_tweets": 20,
  "include_retweets": false,
  "include_replies": false
}
```
