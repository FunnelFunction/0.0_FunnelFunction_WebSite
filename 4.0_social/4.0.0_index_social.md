# Social Aggregation Index

> **4.0_social** - Social media feeds and aggregation engine

---

## Philosophy

Social represents the **top of the content funnel**:
- Highest frequency
- Shortest form
- Presence and engagement
- "I exist" signals

The social section **aggregates** content FROM social platforms back INTO the website, creating a living proof of identity.

## The SEO Strategy

```
┌─────────────────────────────────────────────────────────────┐
│  /social/ page                                              │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  Extracted keywords (CRAWLABLE by Google)           │   │
│  │  "f(Commitment), identity congruence, marketing..." │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌──────────────────┐  ┌──────────────────┐                │
│  │ [LinkedIn Embed] │  │ [Twitter Embed]  │  (Visual)      │
│  └──────────────────┘  └──────────────────┘                │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  More extracted topics (CRAWLABLE)                  │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

Google can't index iframe content, but CAN index the extracted keywords we place around the embeds.
