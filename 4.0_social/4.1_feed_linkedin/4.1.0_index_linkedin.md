## Configuration

```json
{
  "platform": "linkedin",
  "profile_url": "https://linkedin.com/in/funnelfunction",
  "company_url": "https://linkedin.com/company/funnelfunction",
  "api_version": "v2",
  "refresh_interval": "6h",
  "extract_keywords": true,
  "max_posts": 10
}
```
