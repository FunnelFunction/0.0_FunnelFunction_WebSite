# Simulators Index

## Philosophy

Simulators are **interactive educational tools** that let users experience marketing mathematics through play.

They are:
- Hosted on GitHub Pages
- Embedded in WordPress via script/iframe
- Built with React + TypeScript + Vite

Experience how different mental states affect marketing reception.

### Forecast Engine (Prospective)
Plan campaigns with predicted 12-week outcomes.

### Autopsy Lab (Retrospective)
