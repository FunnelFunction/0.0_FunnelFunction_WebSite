# Campaign Commander

> **5.1_simulator_campaign_commander** - Seller POV marketing simulator

---

## Overview

Campaign Commander lets users control marketing variables and see real-time effects on conversion probability.

**Perspective**: Seller/Marketer
**Purpose**: Understand how marketing inputs affect attention capture

---

## Features

- Budget allocation slider
- Creative quality controls
- Targeting precision adjustment
- Brand authenticity meter
- Channel selection
- Real-time score calculation
- AI-generated insights

---

## The Math

```
Attention Score = (B × M × S) / (N + L + Θ)

Where:
- B = Body (sensory strength from creative)
- M = Mind (targeting precision)
- S = Soul (brand-identity alignment)
- N = Noise (competitive clutter)
- L = Load (cognitive burden)
- Θ = Friction (conversion barriers)
```

---

## Controls

| Control | Variable | Range |
|---------|----------|-------|
| Budget | Amplifier | $1K - $1M |
| Creative Quality | B (Body) | 0 - 1.0 |
| Targeting Precision | M (Mind) | 0 - 1.0 |
| Brand Authenticity | S (Soul) | 0 - 1.0 |
| Channel Selection | N, L modifiers | Multi-select |
