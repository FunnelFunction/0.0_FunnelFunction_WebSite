# Commitment Function

> **6.2_equation_commitment_function** - The mathematical replacement for "Trust"

---

## The Equation

```
f(Commitment) = P_Transactional · P_Enduring
```

Where:
- **P_Transactional** = Probability of initial purchase
- **P_Enduring** = Probability of sustained loyalty

---

## Enduring Loyalty Function

```
                    1   t₀+T
P_Enduring(T) = ─── ∫     σ( S(τ) / (Σ_post(τ))^k − β ) dτ
                 T   t₀
```

| Symbol | Definition |
|--------|------------|
| σ(x) | Sigmoid activation function |
| S(τ) | Soul alignment over time |
| Σ_post | Post-purchase friction |
| k | Sensitivity exponent (k ≥ 1) |
| β | Baseline threshold |
| T | Time horizon |

---

## Soul Alignment Decomposition

```
S(τ) = ω_Π · Π(Archetype) + ω_ι · ι(Identity)
```

Where ω_Π + ω_ι = 1

---

## Post-Purchase Friction

```
Σ_post = L_service + F_unforeseen + Θ_support
```

| Component | Description |
|-----------|-------------|
| L_service | Service latency (MTTR) |
| F_unforeseen | Hidden fees, unexpected costs |
| Θ_support | Effort to get help |
